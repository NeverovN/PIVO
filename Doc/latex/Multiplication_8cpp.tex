\doxysubsection{Detailed Description}
Перегруженные методы $\ast$ и $\ast$=. 

Перегруженные методы $\ast$ и $\ast$=, которые дают возможность умножать объекты класса друг на друга В каждом из методов создается объект класса result, который будет хранить в себе ответ, после чего начинается перемножение каждого из членов 1-\/го многочлена на каждый из членов 2-\/го многочлена.

Оператор $\ast$ возвращает объект класса, т.\+к. вызывается не для указателя this, а для одного из множителей.

Оператор $\ast$= же изменяет сам объект, для которого был вызван, поэтому у него нет возвращаемого значения.
